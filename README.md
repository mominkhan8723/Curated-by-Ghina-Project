# Curated by Ghina

Luxury event management website for a premium boutique brand in Islamabad, Pakistan.

## Features

- Cinematic hero section with parallax scrolling
- Elegant about section with editorial layout
- Luxury service cards with glassmorphism effects
- Pinterest-inspired masonry gallery
- Horizontal scrolling experience section
- Testimonials with glass effects
- Instagram-inspired feed section
- Contact form with smooth animations
- Fully responsive design
- Framer Motion animations throughout

## Color Palette

- Warm beige: #EFE7DA
- Ivory cream: #F8F4EE
- Chocolate brown: #5A3E2B
- Soft gold: #C8A97E
- Dusty rose: #D9B6A3

## Typography

- Cormorant Garamond (Serif headings)
- Inter (Sans-serif body)

## Getting Started

```bash
npm install
npm run dev
```

## Building for Production

```bash
npm run build
npm start
```
